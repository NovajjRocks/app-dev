# My Favorite Series: Code Geass

## Overview
Code Geass: Lelouch of the Rebellion is a Japanese anime series that combines mecha action with political intrigue and strategic warfare.

## Why I Love It
- **Complex protagonist**: Lelouch vi Britannia is a brilliantly written anti-hero
- **Strategic battles**: The chess-like tactical warfare is incredibly engaging
- **Geass power system**: The supernatural abilities add unique plot elements
- **Plot twists**: The series is full of shocking revelations and turns

## Favorite Characters
1. Lelouch vi Britannia
2. C.C
3. Jeremiah Gottwald

> "I destroy the world and create it anew."
